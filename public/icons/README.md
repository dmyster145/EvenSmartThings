# Glasses display icons

Place optional PNG overrides here using these exact names:

- **checkmark.png**: shown when all actions succeed
- **error.png**: shown when all actions fail
- **exclamation.png**: shown when results are mixed (partial success)

Each icon is displayed in a `60×60` confirmation container in the lower-right corner.
For best quality, provide source assets already sized to `60×60` so no resampling is needed.
If any file is missing, the app falls back to built-in placeholder graphics.
